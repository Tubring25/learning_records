# 对象的扩展

### 属性的简洁表示法

 ES6 允许在大括号里面，直接写入变量和函数，作为对象的属性和方法。这样的书写更加简洁

### 属性名表达式

定义对象

```jsx
let lastWord = 'last word';

const a = {
  'first word': 'hello',
  [lastWord]: 'world'
};

a['first word'] // "hello"
a[lastWord] // "world"
a['last word'] // "world"
```

属性名表达式与简洁表示法，不能同时使用

属性名表达式如果是一个对象，默认情况下会自动将对象转为字符串`[object Object]`
